Для каждого проекта 12НГ-спринта планируется добавить два маркера на лист "Самооценка":
- Субъективный (1-5): как ощущаю прогресс
- Объективный ("чтобы что"): измеримый результат

| Проект | Субъективный | Объективный | Источник |
|--------|-------------|-------------|----------|
| F1 Сон+питание | Как чувствую утром? | HRV (Apple Watch) | Apple Health |
| F2 Метрики жизни | Понимаю ли данные? | Кол-во автособираемых метрик | Ручной |
| К1 Горы | Заряжает природа? | Среднее шагов/нед | Apple Watch |
| К2 Радости | Кайф от жизни? | Часы на хобби/нед | Ручной |
| Л1 IWE | Двигается проект? | Фаза (0-4) | Ручной |
| Л2 VO2Max | Прогресс в форме? | VO2Max + пульс покоя | Apple Watch |
| С1 ZP/FPF дети | Дети вовлекаются? | Принципов закрыто | Aist-бот |
| С3 Инфра детей | Контроль? | Пунктов чеклиста | WP-3 |
| С4 Отпуск | Предвкушение? | Гейт пройден? | Ручной |
| Ка1 Портфель | Контроль? | % проектов с next action | Day Close |

| Ка3 ₸25 млн | Спокоен? | Накоплено ₸ vs цель | Банк |

**Why:** user хочет видеть не только субъективное ощущение, но и объективную метрику "чтобы что" для каждого проекта
**How to apply:** при следующей итерации трекера добавить столбцы на лист "Самооценка"
