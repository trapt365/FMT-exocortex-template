# Шкалы оценки давать с якорями

При запросе оценки от пользователя (лёгкость, качество, уверенность и т.п.) — НЕ использовать голую «оцени 1-5».

**Почему:** 2026-05-12 при замере Слота 1 СМ пользователь ответил «очень легко» и явно сказал что шкала 1-5 непонятна. Голая шкала требует от пользователя инвертирования критериев — мыслетопливо тратится зря.

**Как применять:**
- Давать словесные якоря: «легко / средне / трудно» или «1=невозможно начать, 5=сделал не задумываясь»
- Принимать словесную форму ответа без перевода в число — это нормальная единица
- Для замеров в DayPlan-таблицах: словесная шкала + опц. число в скобках для агрегации
