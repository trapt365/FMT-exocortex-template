# Навык Алисы «Мой рацион» — подключение (WP-41)

> Часть пользователя, ~30 минут. Код готов: `index.py` (облачная функция),
> `export.py` (экспортёр снимка). Снимок лежит в `DS-strategy/current/alice-ration.json`.

## Как это работает

```
/cronometer → cronometer-remaining.json → export.py --push → GitHub (DS-strategy)
                                                                  ↓
колонка ← Яндекс Диалоги ← облачная функция (читает снимок из GitHub)
```

## Шаг 1. Токен GitHub (5 мин)

1. GitHub → Settings → Developer settings → **Fine-grained personal access tokens** → Generate new token.
2. Имя: `alice-ration`. Repository access: **Only select repositories** → `trapt365/DS-strategy`.
3. Permissions → Repository permissions → **Contents: Read-only**. Больше ничего.
4. Скопировать токен (показывается один раз).

## Шаг 2. Облачная функция (10 мин)

1. Зайти на [console.yandex.cloud](https://console.yandex.cloud) (вход обычным Яндекс-аккаунтом — тем же, что на колонке). При первом входе создастся облако и каталог `default` — соглашаться.
2. Меню → **Cloud Functions** → Создать функцию. Имя: `alice-ration`.
3. Редактор: среда **Python 3.12**, способ — «Редактор кода». Вставить содержимое `index.py` целиком в файл `index.py`.
4. Точка входа: `index.handler`. Таймаут: 5 сек. Память: 128 МБ.
5. Переменные окружения:
   - `GITHUB_TOKEN` = токен из шага 1
   - `GITHUB_REPO` = `trapt365/DS-strategy`
6. Нажать «Сохранить изменения» (создастся версия).
7. Вкладка «Обзор» → включить флажок **«Публичная функция»**.

Проверка прямо в консоли: вкладка «Тестирование» → тип данных «Алиса» (или вставить JSON из `test-event.json`) → Запустить. В ответе должен быть текст сводки.

## Шаг 3. Навык в Диалогах (10 мин)

1. Зайти на [dialogs.yandex.ru/developer](https://dialogs.yandex.ru/developer) тем же аккаунтом → Создать диалог → **Навык в Алисе**.
2. Заполнить:
   - Название: `Мой рацион`
   - Активационное имя: `мой рацион`
   - Backend: выбрать **«Функция в Yandex Cloud»** → указать функцию `alice-ration` (URL не нужен).
   - Доступ: **Приватный** (навык работает только у тебя, без модерации).
   - Остальные поля (иконка, описание) — по минимуму, для приватного не важны.
3. Сохранить → вкладка «Тестирование»: написать что-нибудь в чат, убедиться, что отвечает сводкой.

## Шаг 4. Голосовая проверка на колонке (5 мин)

Колонка должна быть привязана к тому же Яндекс-аккаунту.

- «Алиса, запусти навык мой рацион» → короткая сводка.
- «Сколько осталось» → остаток по пяти показателям.
- «Что съесть» → рекомендация + готовые блюда.
- «Хватит» → навык закрывается.

## Эксплуатация

- Снимок обновляется при каждом прогоне `/cronometer` (расширение `cronometer.alice-export.md`)
  или вручную: `python3 export.py --push`.
- Если данные старше 6 часов, Алиса честно скажет, когда они обновлялись.
- Токен протухнет (или сменится репо) → функция ответит «не смог достать сводку» —
  обновить переменную `GITHUB_TOKEN` в функции.

## Стоимость

Yandex Cloud Functions: бесплатный лимит 1 млн вызовов в месяц — навык на одного
человека не выйдет и за тысячу. Фактически 0 ₸.
